# Frontend Developer Internship Landing Page

A responsive landing page for a frontend developer internship program. The page is built with plain HTML, CSS, and JavaScript.

## Files

```text
frontend-internship-site/
├── index.html
├── styles.css
├── script.js
└── README.md
```

## Features

- Responsive layout for desktop, tablet, and mobile screens
- Fixed navigation bar
- Hero section with internship call-to-action
- Internship highlights with animated counters
- Role snapshot card
- Curriculum / learning tracks section
- Application process timeline
- Apply button with email interaction
- Clean modern UI using HTML, CSS, and JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How To Run

Open the `index.html` file in any modern web browser.

No installation or build step is required.

## Sections Included

1. Hero Section
2. Role Snapshot
3. Learning Tracks
4. Application Process
5. Final Call To Action

## JavaScript Functionality

The `script.js` file handles:

- Active navigation link highlighting
- Counter animation for internship statistics
- Apply button message and email redirect

## Customization

You can update the internship details by editing:

- Text content in `index.html`
- Colors and layout in `styles.css`
- Button behavior in `script.js`

## Preview

Open:

```text
index.html
```

in your browser to view the landing page.
